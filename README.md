# 🚀 RideQ Frontend

Frontend for RideQ — an Uber-like ride booking system.

## 🎯 Features

- User & Driver interfaces
- Map integration using Leaflet.js
- Pickup & drop selection
- Trip creation & status tracking
- Polling-based updates

---

## ⚙️ Tech Stack

- HTML
- CSS
- JavaScript
- Leaflet.js

---

## 🚀 How to Run

1. Start backend server (Spring Boot)
2. Open `login.html` in browser

---

## ⚠️ Note

Make sure backend is running at:

http://localhost:9090
